# NDA-Safe Scope Statement

This repository is a demonstration package. It avoids proprietary source code, real client data, licensed ephemeris files, and commercial secrets.

It is designed to show how I would structure the work after mutual NDA and codebase access:

- Audit current computation flow.
- Reproduce non-determinism.
- Lock input normalization and timezone handling.
- Validate output against Swiss Ephemeris native data.
- Cross-check selected cases with AstroSeek and Astrodienst.
- Build milestone acceptance reports with tolerances and diffs.

The real project implementation would be performed inside the client's Lovable/Supabase/React codebase or through a GitHub sync, depending on the founder's preference.
